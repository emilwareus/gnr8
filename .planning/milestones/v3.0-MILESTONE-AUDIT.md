## Cross-Phase Integration — 5/5 Wired

1. Shared SDK planning facts now feed all Go, Python, and TypeScript emitters for operation, schema,
   grouping, docs metadata, auth, error, runtime, pagination, package, and file-plan decisions.
2. OpenAPI 3.1 output, generated SDK methods, generated docs, and `gnr8 compat` share stable operation and
   grouping facts.
3. Auth, typed errors, retries/timeouts/idempotency, and hooks are graph-driven rather than target-specific
   template guesses.
4. SDK readiness and package metadata are validated locally through `doctor` and target-language checks
   where toolchains are available.
5. Operation docs/examples/error responses and common request media types are represented at the graph/SDK
   boundary instead of being hand-coded in one target.
