Type encoding (adjacently tagged `{"type": "<variant>", "of": <payload>}`):
- primitive â†’ `{"prim":"string"}` | `{"prim":"bool"}` | `{"prim":"int","bits":64,"signed":true}` | `{"prim":"float","bits":64}` | `{"prim":"bytes"}`. Python `int`â†’int64 signed; `float`â†’float64. (RESEARCH Pitfall 2.)
- well_known â†’ bare snake_case string payload: `{"type":"well_known","of":"uuid"}`.
- array â†’ `{"type":"array","of":<Type>}`.
- map â†’ `{"type":"map","of":{"key":<Type>,"value":<Type>}}`.
- named â†’ bare id string: `{"type":"named","of":"app.models.Author"}`.
- object â†’ `{"type":"object","of":[<FieldFact>...]}`.
- enum â†’ SORTED string array: `{"type":"enum","of":["asc","desc"]}` (enum.Enum VALUES + Literal strings, both sorted; RESEARCH Pitfall 4).
- union â†’ array of Type, SOURCE ORDER preserved (NOT sorted; RESEARCH Pitfall 5).
- any â†’ `{"type":"any","of":{}}` (empty object, never null/bare-string).

FieldFact keys EXACTLY: `json_name, required, optional, nullable, schema, description, example`.
`description`/`example` are ALWAYS null here. Four-axis rules (RESEARCH Pitfall 3):
- optional = field has a default (`= x`, `field(default=...)`, `field(default_factory=...)`).
- nullable = type admits None (`Optional[T]` or `T | None`).
- required = not optional. nullable does NOT affect required.

SchemaFact keys: `id, name, body, span`. id = `<dotted-module>.<ClassName>` (relpath, `/`â†’`.`, drop `.py`).
SourceSpan keys: `file, start_line, end_line` â€” file is the CANONICAL ABSOLUTE path `os.path.join(canonical_target_dir, relpath)` (host relativizes; RESEARCH Pattern 2/C).
DiagnosticFact keys EXACTLY: `severity, message, file, line` (line is a single int, NOT a span).
Top-level GoFacts keys: `module, routes, schemas, diagnostics`. module = `basename(canonical_target_dir)` (RESEARCH Pattern 3).

From goextract/main.go (orchestration to mirror in pyextract `run()`):
`load â†’ diag.New() â†’ schemas â†’ module â†’ routes â†’ assemble GoFacts â†’ facts.Marshal`. Routes are
EMPTY in this plan (recognizers land in Plan 03/04); emit `"routes": []`.

From goextract/internal/facts/facts.go (Marshal/sortDoc discipline to mirror):
schemas by id; object fields by json_name; enum members lexically; diagnostics by (file,line,message);
routes by (path,method); params by (name,location); responses by status. Union members NOT sorted.
