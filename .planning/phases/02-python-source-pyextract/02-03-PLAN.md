FunctionDef whose decorator_list holds a Call with func = Attribute(value=Name("router"), attr="get"),
args = [Constant("/")], keywords = [keyword("response_model", Name("ListBooksResponse")),
keyword("status_code", Constant(201))].
- method = attr.upper() for attr in {get,post,put,delete,patch,head,options}.
- path = the first Constant arg, VERBATIM (`/`, `/{book_id}` already brace form — keep).
- `APIRouter(prefix="/books")` is an Assign; the prefix is recorded SEPARATELY and NEVER folded into the path (rule 1; snapshot base_path stays `/`).
- params from the typed FunctionDef signature. A param whose name appears in the path template (e.g. `book_id: int`) → location path, required true, int→int64. Otherwise → location query; required = (no default). `Optional[Enum]` param → schema is a `named` ref to the enum schema (RESEARCH OQ3), required false.
- request_body: a parameter whose annotation resolves to a BaseModel/@dataclass class → `{"ref_id": "<id>"}`.
- response: `response_model=` → response body ref; `status_code=` Constant → status; default 200 when absent.
